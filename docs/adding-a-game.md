# Как добавить игру

1. Откройте `data/games.json`.
2. Добавьте объект игры в массив:

   ```json
   {
     "id": "new-game-id",
     "name": "Полное название",
     "shortName": "Короткое",
     "initials": "NG",
     "publisher": "Издатель",
     "colorFrom": "#60a5fa",
     "colorTo": "#1d4ed8"
   }
   ```

3. Требования:
   - `id` — уникальный слаг в kebab-case, без пробелов. Он используется в URL (`/games/<id>`) и в поле `gameId` записей.
   - `initials` — 2–5 символов для аватара-заглушки (логотипы игр не используем).
   - `colorFrom` / `colorTo` — два hex-цвета для нейтрального градиента.
4. (Опционально) добавьте официальный источник в `data/sources.json` с тем же `gameId`.
5. Запустите проверки:

   ```bash
   npm run validate
   npm run lint && npm run typecheck && npm run build
   ```

6. Игра появится на странице `/games` автоматически. Записи к ней добавляются отдельно — см. [adding-records.md](adding-records.md).

> Не добавляйте записи с выдуманными датами только ради наполнения. Реальные данные берите из официальных источников.
