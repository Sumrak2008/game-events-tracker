# Проверка и публикация обновлений

Данные живут в JSON-файлах репозитория. Обновление = правка JSON → проверки → коммит в GitHub → автодеплой на Vercel.

## 1. Локальные проверки (обязательно)

Запустите по очереди. Все должны пройти:

```bash
npm run format      # Prettier (форматирование)
npm run lint        # ESLint
npm run typecheck   # TypeScript (tsc --noEmit)
npm run validate    # валидация JSON-данных
npm run build       # production-сборка
```

Быстрый агрегат без сборки: `npm run check`.

**Если любая проверка падает — не публикуйте.** Сначала исправьте причину.

## 2. Отчёт об изменениях

Перед публикацией подготовьте короткий отчёт:

- какие записи **добавлены** (игра, тип, заголовок, даты, источник);
- какие **изменены** и почему;
- какие **перешли в архив** (по прошедшему `endAt`);
- подтверждение, что реальные записи имеют `sourceUrls`, `sourceType`, `confidence` и дату проверки `verifiedAt`.

Покажите отчёт и дождитесь подтверждения, прежде чем пушить.

## 3. Публикация на GitHub

> Выполняется как отдельный этап, после ревью приложения.

```bash
git add -A
git commit -m "data: обновление баннеров/событий/сезонов"
git push
```

Если репозиторий ещё не создан:

```bash
git init
git add -A
git commit -m "Initial commit"
gh repo create game-events-tracker --public --source . --push
```

## 4. Деплой на Vercel

- Импортируйте репозиторий на https://vercel.com (New Project → Import).
- Framework определяется автоматически как **Next.js**, дополнительная настройка не нужна.
- Каждый push в основную ветку запускает новый деплой; данные из JSON попадают в сборку и видны на всех устройствах.

## Чек-лист перед публикацией

- [ ] `npm run validate` — без ошибок
- [ ] `npm run lint`, `npm run typecheck`, `npm run build` — без ошибок
- [ ] Реальные записи имеют `sourceUrls`, `sourceType`, `confidence` и `verifiedAt`
- [ ] Даты в ISO 8601 с часовым поясом, `region` и `timezone` заполнены
- [ ] Завершённые записи не удалены
- [ ] Подтверждённые данные не заменены догадками
- [ ] Подготовлен отчёт об изменениях
